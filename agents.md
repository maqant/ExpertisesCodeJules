# Instructions pour les Agents IA (Jules)

## 🎯 Principe Fondamental : Minimalisme Chirurgical
Ton objectif est de remplir la tâche demandée avec le **minimum de modifications nécessaires**. 

1. **Isolation :** Ne modifie, ne supprime et ne déplace aucun code qui n'est pas directement impliqué dans la résolution du ticket. 
2. **Respect de l'Existant :** Tout code présent est là par nécessité. Ne présume jamais qu'une fonction est inutile.


## 🛠 Stack Technique
* **React / Vite / Tailwind**
* **Zustand** (`src/store/financeStore.js`) pour la logique métier.
* **React Context** (`src/context/ExpertiseContext.jsx`) pour le dossier.

## 🚀 Workflow
* Une seule tâche à la fois.
* Commits clairs et fonctionnels directement sur `main`.
